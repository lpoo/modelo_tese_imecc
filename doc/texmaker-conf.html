Configuração do Texmaker (versão 3.2) para uso do do modelo de dissertação/tese do IMECC

Ao abrir o Texmaker, você encontrará uma janela semelhante a figura abaixo.

Verifique na barra inferior (de status) se a codificação utilizada é a UTF-8 (ver figura acima). Se a codificação utilizada for a UTF-8, você pode ir para a seção Configuração do documento mestre.

Configuração da codificação

Caso a codificação utilizada seja outra que não a UFT-8, selecione "Opções" -> "Configurar o Texmaker" e irá aparecer uma janela semelhante a figura abaixo.

Na aba "Editor", selecione UTF-8 para o parâmetro "Codificação". Depois pressione o botão "OK".

Configuração do documento mestre

O documento mestre do modelo de dissertação/tese do IMECC é tese.tex. Para facilitar o uso do modelo, recomenda-se que o editor seja configurado para tratar o arquivo tese.tex como o documento mestre.

Para configurar o documento mestre, abra o arquivo tese.tex como o Texmaker, selecione "Opções" na barra superior e, depois, "Definir o documento atual como 'Documento Mestre'" (ver figura abaixo).

O Texmaker não possui a funcionalidade de projeto, logo toda vez que for utilizar o modelo você terá que configurar o documento mestre.
