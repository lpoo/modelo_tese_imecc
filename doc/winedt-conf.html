Configuração do WinEdt (versão 7.1) para uso do do modelo de dissertação/tese do IMECC

O WinEdt tem o UTF-8 como codificação padrão, portanto não é necessário nenhuma modificação para o uso do modelo de dissertação/tese do IMECC.

A instalação do WinEdt 7.1 com MiKTeX e TeX Live é automática, ou seja, ao baixar e instalar o editor ele automaticamente encontra o caminho da pasta contendo o LaTeX.

Caso, queira mudar as configurações para o caminho do LaTeX e visualizadores entre em "options" -< "Execution Modes" -< "PATH" e para alterar visualizadores de PDF entre em "options" -< "Execution Modes" -< "PDF viewer". Para mais detalhes entre em "http://www.winedt.com/index.html.
